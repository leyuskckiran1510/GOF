# Conway's Game of Life in cli-engine

Conway's Game of life in terminal using [cli-engine](https://github.com/leyuskckiran1510/cli-engine.git)

# Get Started

```bash
git clone https://github.com/leyuskckiran1510/cli-engine.git
cd cli-engine
make
```
